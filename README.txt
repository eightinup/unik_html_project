Nova Academy — учебный HTML/CSS/JS проект

Состав проекта:
- index.html — главная страница
- courses.html — страница курсов
- teachers.html — страница преподавателей
- pricing.html — страница тарифов
- register.html — страница регистрации
- style.css — общие стили сайта
- script.js — интерактив: меню, фильтры, тарифы, валидация формы

Как запустить:
1. Скачайте папку проекта или архив.
2. Откройте файл index.html в браузере.
3. Для навигации используйте меню сайта.

Особенности:
- без Bootstrap
- адаптивная верстка
- современный дизайн
- чистый HTML, CSS и JavaScript
